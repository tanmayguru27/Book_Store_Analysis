# Bookstore Management Database

This project demonstrates the design and implementation of a relational database for a bookstore using MySQL. The database consists of Books, Customers, and Orders tables connected through primary and foreign key relationships. Data was imported from CSV files and analyzed using SQL queries to generate business insights such as total revenue, inventory status, customer purchasing behavior, genre-wise sales, frequently ordered books, and remaining stock after order fulfillment.

## Skills Demonstrated

* Database Design & Normalization
* SQL Query Writing
* Joins (INNER JOIN, LEFT JOIN)
* Aggregate Functions (SUM, COUNT, AVG)
* Filtering & Sorting (WHERE, ORDER BY)
* Grouping & Analysis (GROUP BY, HAVING)
* CSV Data Import & Management
* Inventory and Sales Analytics
